# Python_netmiko

Description:
------------
Connect to network devices via ssh or telnet.
First,this script will try to connect to network device via ssh,In case of failure it will try to connect via telnet. 

Rrequirements: 
------------
[+] netmiko module

Usage: 
------
[+] In the script change USER and PASSWORD variables.
[+] To run commands on several devices just put the ip's in Devices ip's.txt file in the same folder as the script.
[+] To run commands on only one device read the comment in lines 122-123.

Enjoy, and remember netmiko is Easylife with this wonderful script.
